# COW MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现一个纯库级、正确性优先的 `COW`（Copy-On-Write，写时复制）事务 `MVP`，支持根事务提交/回滚、栈式 `Savepoint`、组件级受控写入和基础 benchmark。

**Architecture:** 以 `RuntimeRoot` 与组件为聚合数据边界，使用内存 `Store` 保存已提交根状态，`Run` 在一次处理作用域内创建 `TxSession`，通过 overlay（覆盖层）与检查点栈管理未提交修改。第一版只提供手写示例组件与纯库级接入，不实现宿主适配层或代码生成器。

**Tech Stack:** Go 1.25、标准库 `context`/`sync/atomic`/`maps`/`testing`

# COW MVP 运行时设计

## 1. 目标

本设计仅覆盖 `MVP`：

- `Run(ctx, store, fn)` 是唯一标准入口；
- 支持根事务提交/回滚；
- 支持多层栈式 `Savepoint`；
- 仅支持组件级受控写入；
- 提供简版 `DirtySet`；
- 不实现宿主适配层、代码生成器、手动事务绑定接口。

## 2. Store 契约

```go
type Store[T any] interface {
    Load() *T
    Commit(next *T)
}
```

`Store` 只承担已提交根状态的读取与替换，不承担锁、事务嵌套或持久化职责。

## 3. Session 形态

```go
type TxSession[T any] struct {
    store       Store[T]
    base        *T
    work        *T
    checkpoints []checkpoint[T]
    dirty       DirtySet
    committed   bool
    finished    bool
}
```

`work` 表示当前事务视图；检查点以快照克隆函数产生的根快照为单位管理，`MVP` 先保证正确性，不强求检查点阶段仍是最优局部复制。

## 4. 受控写入模型

- 业务代码不直接修改 `store.Load()` 返回值。
- 示例组件通过手写访问器确保首次写入时拷贝组件及其可变字段。
- `Savepoint` 后的回滚通过恢复根快照与 `DirtySet` 快照实现。

## 5. 错误模型

- 缺失事务上下文：返回 `ErrNoSession`
- 已结束事务继续操作：返回 `ErrSessionClosed`
- 非法检查点：返回 `ErrInvalidSavepoint`
```

```go
package cow

import (
    "context"
    "errors"
    "sync/atomic"
)

var (
    ErrNoSession     = errors.New("cow: no active session")
    ErrSessionClosed = errors.New("cow: session already finished")
)

type Store[T any] interface {
    Load() *T
    Commit(next *T)
}

type memoryStore[T any] struct {
    ptr atomic.Pointer[T]
}

func newMemoryStore[T any](root *T) *memoryStore[T] {
    s := &memoryStore[T]{}
    s.ptr.Store(root)
    return s
}

func (s *memoryStore[T]) Load() *T {
    return s.ptr.Load()
}

func (s *memoryStore[T]) Commit(next *T) {
    s.ptr.Store(next)
}

type sessionKey struct{}

type TxSession[T any] struct {
    store    Store[T]
    base     *T
    work     *T
    finished bool
}

func FromContext[T any](ctx context.Context) (*TxSession[T], bool) {
    sess, ok := ctx.Value(sessionKey{}).(*TxSession[T])
    return sess, ok
}

func (s *TxSession[T]) Mutable(pick func(root *T) any) any {
    return pick(s.work)
}

func Run[T any](ctx context.Context, store Store[T], clone func(*T) *T, fn func(context.Context) error) (err error) {
    base := store.Load()
    sess := &TxSession[T]{
        store: store,
        base:  base,
        work:  clone(base),
    }
    txCtx := context.WithValue(ctx, sessionKey{}, sess)
    defer func() {
        sess.finished = true
        if r := recover(); r != nil {
            panic(r)
        }
    }()
    if err = fn(txCtx); err != nil {
        return err
    }
    store.Commit(sess.work)
    return nil
}
```

import "errors"

type SavepointID uint64

type checkpoint[T any] struct {
    id    SavepointID
    root  *T
    dirty DirtySet
}

- [ ] **Step 2: 扩展 `TxSession` 并实现 `Savepoint`/`RollbackTo`**

```go
type TxSession[T any] struct {
    store       Store[T]
    base        *T
    work        *T
    checkpoints []checkpoint[T]
    nextID      SavepointID
    dirty       DirtySet
    finished    bool
}

func Savepoint[T any](ctx context.Context) (SavepointID, error) {
    sess, ok := FromContext[T](ctx)
    if !ok {
        return 0, ErrNoSession
    }
    cp := checkpoint[T]{
        id:    sess.nextID + 1,
        root:  sess.clone(sess.work),
        dirty: sess.dirty.Clone(),
    }
    sess.nextID = cp.id
    sess.checkpoints = append(sess.checkpoints, cp)
    return cp.id, nil
}

func RollbackTo[T any](ctx context.Context, id SavepointID) error {
    sess, ok := FromContext[T](ctx)
    if !ok {
        return ErrNoSession
    }
    n := len(sess.checkpoints)
    if n == 0 || sess.checkpoints[n-1].id != id {
        return ErrInvalidSavepoint
    }
    cp := sess.checkpoints[n-1]
    sess.checkpoints = sess.checkpoints[:n-1]
    sess.work = sess.clone(cp.root)
    sess.dirty = cp.dirty.Clone()
    return nil
}
```
